Назва датасету: DNA
Назва цільової змінної: Class
Який клас вважався позитивним: ei (exon/intron)
Які змінні були використані як предиктори: V1-V180 (бінарні індикатори нуклеотидів)
Які змінні були виключені та чому: Жодна змінна не виключалась, оскільки всі ознаки є релевантними.
Які моделі класифікації були побудовані: Логістична регресія, Дерево рішень, Випадковий ліс, SVM (базовий та оптимізований).
Яка модель виявилася найкращою: Random Forest (за збалансованістю Recall та AUC) та базова SVM (за найвищою Accuracy).

Основні результати:
У ході роботи було виконано бінаризацію багатокласового датасету DNA для прогнозування класу 'ei' (межа екзон/інтрон). Оригінальні 60 нуклеотидів були перетворені на 180 бінарних ознак, тому додаткове масштабування не потребувалось. Спостерігався певний дисбаланс класів (клас 'ei' складає меншість). Усі побудовані моделі показали дуже високу точність (Accuracy > 94%). Оскільки в цій генетичній задачі найбільш небезпечною є помилка False Negative (пропуск справжньої межі), найважливішою метрикою є Recall. Модель Random Forest показала найвищий Recall (0.9600) та площу під кривою AUC (0.9927), тому саме її найдоцільніше використовувати на практиці. Водночас базова модель SVM продемонструвала найвищу Accuracy (0.9749) та Precision (0.9848).
